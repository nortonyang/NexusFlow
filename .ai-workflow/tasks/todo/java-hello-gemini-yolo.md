# Task: Create A Simple Java Program With File Tool

## Goal

Create a simple Java program for this repository.

## Required Change

Create this file:

```text
examples/java/HelloGemini.java
```

The Java program must:

- Define a public class named `HelloGemini`.
- Include a `public static void main(String[] args)` method.
- Print exactly this line:

```text
Hello from Gemini Java workflow!
```

## Constraints

- Do not inspect unrelated repository files.
- Do not modify existing files.
- Only create `examples/java/HelloGemini.java`.
- Use the available file editing tool directly.
- Keep the code minimal and compilable with `javac`.
